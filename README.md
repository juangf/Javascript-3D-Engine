## Camera controls
- Key `W`: Front.
- Key `A` or `←`: Left.
- Key `D` or `→`: Right.
- Key `S`: Back.
- Key `↓`: Down.
- Key `↑`: Up.
- Key `C`: Restart camera position.
- Key `1`: Default scene.
- Key `2`: Pyramid scene.
- Mouse Pointer: LookAt.
